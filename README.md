# Youtube-Buttons-Controller-Firefox-Extention

**YouTube Button Controller**

*This browser extension simplifies your YouTube experience on Firefox, making it easy for you to control playback, mute, and navigation buttons.*

**How to Use:**
1. Open the YouTube tab you want to control.
2. Click on 'Get Tab' to identify the tab the plugin should control.
3. Use the provided control buttons to manage playback, mute, and navigation.
4. Enjoy it :)
